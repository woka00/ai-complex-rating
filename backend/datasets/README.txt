Папка для пользовательских датасетов.

Сюда сохраняются датасеты, загруженные через интерфейс на вкладке «Тестирование»
или через API POST /api/datasets/upload.

Структура:
  datasets/
    <название_датасета>/
      images/          — для YOLO: .jpg/.png файлы для прогона детекции
      texts.csv        — для NLP: CSV с колонками text,label (label = pos/neg/neu)
      meta.json        — автоматически создаётся при загрузке (тип, размер, дата)

При запуске тестирования можно выбрать датасет через UI; если не выбран,
используются встроенные тестовые данные (ASSETS из ultralytics для YOLO,
вшитые тексты для NLP).
