Главная страница

Платформа комплексной оценки моделей искусственного интеллекта

Решает фундаментальную проблему выбора ИИ-модели: сегодня этот выбор часто субъективен или ограничен одной метрикой (accuracy, F1, mAP). Наша платформа сравнивает модели по трём группам критериев — точность, устойчивость, контекстная согласованность — с пользовательскими весами и формирует интегральный коэффициент качества K ∈ [0..1] с объясняемым результатом.

Для кого предназначен

🔬

Исследователи и ML-инженеры

Сравнение архитектур моделей под конкретные задачи. Объективное обоснование выбора через интегральный показатель.

🏢

Корпоративные команды

Выбор оптимальной модели под прикладной сценарий с учётом бизнес-ограничений: скорость, размер, стоимость.

🎓

Преподаватели и аналитики

Формализация процесса экспертной оценки. Воспроизводимые результаты и автоматический отчёт.

Ключевые возможности

📊

Взвешенная оценка

Формула K = S_k / S_max по ТЗ. Поддержка любого количества критериев и моделей.

🔬

Локальное тестирование

Запуск моделей детекции прямо в браузере. Прогресс-бар, ETA, автоматическая нормировка метрик.

📈

Анализ чувствительности

Изменение веса критерия → пересчёт K. Видно сразу: устойчив ли выбор или лидер меняется.

📋

Объяснимый отчёт

Не просто "K=0.85" — а почему конкретная модель победила и при каких условиях выбор изменится.

Локальный запуск моделей детекции

Реальный прогон моделей детекции на тестовых изображениях. Метрики: confidence, FPS, устойчивость к шуму. Веса хранятся в backend/models/detection/. Требует: pip install ultralytics opencv-python

Веса скачиваются автоматически при первом запуске
Будет использован: встроенный датасет

Локальный запуск NLP-моделей (sentiment analysis)

Тестирование моделей анализа тональности. Локальные модели грузятся из backend/models/nlp/<имя>/ (если есть), иначе скачиваются с HuggingFace. Метрики: accuracy, robustness (зашумлённые тексты), скорость. Требует: pip install transformers torch

Локальные модели — мгновенно; HF — скачиваются при первом запуске (~30–200 МБ)
Будет использован: встроенный датасет
